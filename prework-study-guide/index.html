Prework Study Guide

HTML

  • The head element contians information about the webpage
  • The header element sits at the top of the webpage and can contain things like logos, titles and a nav bar
  • The body element represents the visible content shown to the user
  • The footer element sits at the bottom of the webpage and may contain things like logos and copyright

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Declarations contain the property we want to apply, and the value of the property
  • Margin, border, padding and content are known as parts of the 'Box Model'

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages changes made in code
  • git commit -m "details": commit changes to git
  • git pull origin main: pulls source code from github to ensure we have the latest commits and changes
